# Gradio Image Viewer

更新日: 2026-04-25

## 概要

`Gradio Image Viewer` は、ローカルフォルダ内の画像をページング付きギャラリーで表示し、クリックでライトボックス（全画面表示）できるサンプルアプリです。

## 主な機能

- フォルダ指定ドロップダウン（候補選択 + テキスト入力）
- `..` による親フォルダ移動
- 絶対パス/相対パス入力対応
- 表示件数指定（デフォルト 10）
- 前ページ / 現在ページ / 次ページ
- サムネイルクリックでライトボックス表示（ズーム・パン・前後移動）

## 動作要件

- Python 3.10 以上推奨
- Windows / macOS / Linux
- ブラウザ: Chrome / Safari / Edge

## セットアップ

```bash
pip install -r requirements.txt
```

## 起動方法

```bash
python app.py
```

起動後、表示された Gradio URL をブラウザで開いて操作してください。

## 使い方

1. `フォルダ選択` で対象フォルダを指定（必要ならテキスト入力）。
2. `表示件数` を設定。
3. `表示` ボタンを押してギャラリーを読み込み。
4. 画像をクリックしてライトボックス表示。

## ファイル構成

- `app.py`: メインUI、フォルダ移動、ページング、ギャラリー表示
- `lightbox.py`: ライトボックスCSSとJS注入ヘルパー
- `lightbox.js`: ライトボックス本体ロジック
- `requirements.txt`: 依存ライブラリ
