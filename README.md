# Gradio Image Viewer

更新日: 2026-04-26

## 概要

`Gradio Image Viewer` は、ローカルフォルダ内の画像をページング付きギャラリーで表示し、クリックでライトボックス（全画面表示）できるサンプルアプリです。  
ギャラリーには **128×128px** の軽量サムネイルを表示し、ライトボックスでは **元画像** を表示します。

## 主な機能

- フォルダ指定ドロップダウン（候補選択 + テキスト入力）
- `..` による親フォルダ移動
- 絶対パス/相対パス入力対応
- 表示件数指定（デフォルト 10）
- 操作行: `表示件数` / `表示` / `前ページ` / `ページ情報` / `次ページ` を同一行に配置（狭い幅では横スクロール）
- `前ページ`〜`次ページ` は `Group` で枠付きのナビゲーションとして表示
- サムネイルクリックでライトボックス表示（ズーム・パン・前後移動）
- ギャラリーは軽量サムネイル表示（ページ単位で生成・キャッシュ）
- ライトボックスは元画像を表示（サムネイルとは別。hidden の JSON で現在ページの元パスを JS に渡す）
- ギャラリー領域の高さは **中身の高さに追従**（ただし画面下端を上限としてクリップ）
- 設定ファイル `config.json` に前回フォルダと前回表示件数を保存/復元

## 動作要件

- Python 3.10 以上推奨
- Windows / macOS / Linux
- ブラウザ: Chrome / Safari / Edge

## セットアップ

```bash
pip install -r requirements.txt
```

## 起動方法

```bash
python app.py
```

起動後、表示された Gradio URL をブラウザで開いて操作してください。

## 使い方

1. `フォルダ選択` で対象フォルダを指定（必要ならテキスト入力）。
2. `表示件数` を設定。
3. `表示` ボタンを押してギャラリーを読み込み。
4. サムネイルをクリックしてライトボックス表示。

## 共有 URL（`share=True`）について

`app.py` は `demo.launch(share=True, allowed_paths=["/"])` で起動します。  
ライトボックスが元画像を `/file=...` で読み込むため、**共有時に絶対パス参照が拒否されないように**しています。

- **注意**: `allowed_paths=["/"]` は広い許可です。運用環境に合わせて、必要なディレクトリだけに絞ることを推奨します（例: 画像出力先のルートのみ）。

## 仕様メモ

- サムネイル生成は「フォルダ内全件一括」ではなく「現在ページ分のみ」です。
- 生成済みサムネイルは OS の一時ディレクトリ配下 `gradio-image-viewer-thumbs/` にキャッシュされます。
- フォルダと表示件数は操作時に `config.json` へ保存され、次回起動時に復元されます。
- ギャラリーの段間は詰めています（外周の上下だけ少し余白）。ギャラリー枠の高さは DOM の実高さに合わせて伸縮します。

## ファイル構成

- `app.py`: メインUI、フォルダ移動、ページング、ギャラリー表示、ギャラリー高さ調整用JS
- `config_store.py`: `config.json` のロード/セーブ管理（クラス実装）
- `lightbox.py`: ライトボックスCSSとJS注入ヘルパー
- `lightbox.js`: ライトボックス本体ロジック（現在ページの元画像パスを参照）
- `config.json`: 前回フォルダ/表示件数（初回起動時は未作成）
- `requirements.txt`: 依存ライブラリ
